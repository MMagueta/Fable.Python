# Fable Python on Django

A minimal example (single file).

## Credits

https://github.com/rnevius/minimal-django

## Install Dependencies

```sh
> dotnet tool restore
> dotnet restore

> pip3 install -r requirements.txt
```

## Build

```
> dotnet fable-py
```

## Run

```sh
> python3 program.py runserver
```

Visit http://127.0.0.1:8000/
